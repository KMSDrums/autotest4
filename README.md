# Autotest - Logout 

## Tech stack:
- [python 3.9] (https://www.python.org)
- [playwright](https://playwright.dev)


## Description:

This autotest is used to check the user's authorization on the site 'Cyber.bet':

1. Login on site

2. Log out from site

3. Check log out

## How to use:

Install Python 3.9

Install Playwright for Python:

`pip3 install playwright`
